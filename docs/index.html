Features

Everything you need to build AI-powered applications

Terminal UI

Full-featured chat client built with BubbleTea. Streaming responses, tool activity display, and beautiful Lipgloss styling. Your AI, your terminal.

./bin/conduit tui --token "conduit_v1_..."
🗝

SSH Access

Remote TUI access via Wish library. Connect from anywhere with SSH key authentication. Secure, encrypted, and instant.

ssh -p 2222 user@your-server.com
✉

Telegram Bot

Native Go adapter with full Bot API support. Photo handling, inline keyboards, and seamless user pairing. Chat with your AI from anywhere.

./bin/conduit pairing create --telegram-id 123456789
⚙

Tool Registry

18 built-in tools with sandbox execution. File editing, web search, vision, scheduling, and more. Extensible architecture for custom tools.

./bin/conduit tools list
🔍

Search & Memory

FTS5 full-text search over documents and messages. Semantic memory search across MEMORY.md and session history. Hybrid Anthropic + Brave web search.

SELECT * FROM fts_documents WHERE content MATCH 'query';

Automation

Chain workflows with dependencies and variables. Cron scheduling for recurring tasks. Heartbeat loop for automated HEARTBEAT.md processing.

Quick Start

Up and running in 60 seconds

terminal 
# Build
$ make build

# Create auth token
$ ./bin/conduit token create --client-name "my-client"

# Start server
$ ./bin/conduit server

# Launch terminal UI
$ ./bin/conduit tui --token "conduit_v1_..."

See the Getting Started Guide for detailed setup instructions.
